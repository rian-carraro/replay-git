# RELEMBRANDO COMANDOS Git/GitHub

## configurações adiciona o nome do desenvolvedor:
````bash
git config --global user.nome "Rian Carraro Permonian"
````

Adiciona email do desenvolvedor : 
````bash
git config --global user.email riancarraro@gmail.com 
````
## Comando Git

Inicia um repositorio
````bash
git init 
````

Adicionar TUDO que tem na pastas 
````bash
git add .
````

Mostra todo status do que foi feito
````bash
git commit -m "nome do commit"
````

Adiciona on-line tudo que foi feito localmente 
````bash
git push
````
